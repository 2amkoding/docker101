## linux file structure:
root

/
    -bin: binaries or programs
    -boot: files related to booting
    -dev: devices
    -etc: editable text configs
    -home: home directories for users
    -root: root user home dir
    -lib: library files, software dependencies
    -var: files updated frequently, locked files, app data
    -proc: running processes
